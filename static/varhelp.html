Помощь

Для изменения свойств плинта нужно ввести его имя (отображается символом "~" и розовым цветом).

Для добавления плинта нужно ввести новое имя (отображается символом "+" и желтым цветом).

Нумерация пар плинтов устанавливается только для добавляемых плинтов, если не установлен "Apply for all".

В групповых операциях можно использовать переменные, указанные в таблице.


Переменные

В именах плинтов
%числосчетчик в именах плинтов
В именах пар
%0текущие
%1из редактора. Каждый плинт с новой строки, разделитель между парами - Tab (формат Excel)
В общих сведениях
%0текущие
%1кросс (на другом конце кабеля)
%2вертикаль ...
%3плинт ...
